## Fundamentals

-   authentication with firebase;
-   save data with firestore;
-   upload with firestorage.

## Libraries

firebase (@8.3.2) & react-router-dom.

## Start server

Set up your firebase config _("./src/config/firebaseConfig.js")_ and execute

```bash
  npm install
  npm run start
```
